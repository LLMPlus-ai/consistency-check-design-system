**FilterChip** — pill toggle for the findings filter row. Pairs a status dot + count with the verdict label. Active fills ink.

```jsx
<FilterChip active count={12}>All</FilterChip>
<FilterChip dotColor="var(--verified)" count={7}>Verified</FilterChip>
<FilterChip dotColor="var(--fabricated)" count={3}>Fabricated</FilterChip>
```
